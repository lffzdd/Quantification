# 量化交易系统学习计划

## 🎯 目标

从零开始，系统学习量化交易，每一天一个主题，配有**代码实践** + **详细讲解**。最终目标是能独立开发、回测、评估一个完整的量化策略。

## 🚀 教学方式

每一天的代码文件都包含：
1. **概念讲解**（注释中详细说明原理）
2. **公式推导**（关键数学公式用注释写清楚）
3. **代码实现**（可直接运行）
4. **思考题**（帮助深入理解）
