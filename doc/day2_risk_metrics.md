# Day 2：收益率与风险度量

> 量化交易的核心不是"赚多少"，而是"每承担一单位风险能赚多少"。
> 今天我们学习如何精确地度量收益和风险。

---

## 一、为什么要学这些指标？

假设有两个策略：

- 策略 A：年化收益 30%，但最大回撤 60%
- 策略 B：年化收益 15%，但最大回撤 8%

哪个更好？如果你只看收益率，一定选 A。但实际中，经历 60% 的回撤意味着你的 100 万变成了 40 万——
大多数人在这个时候就割肉了，根本等不到反弹。

**风险调整后的收益**才是量化交易员真正关心的指标。今天我们就来学这些工具。

---

## 二、收益率：简单收益率 vs 对数收益率

### 2.1 简单收益率 (Simple Return)

**公式：**

$$
R_t = (P_t - P_{t-1}) / P_{t-1} = P_t / P_{t-1} - 1
$$

**含义**：今天的价格相对于昨天涨了百分之多少。

**例子**：股票从 100 元涨到 105 元

$$
R = (105 - 100) / 100 = 0.05 = 5
$$

**代码**：

```python
returns = df["Close"].pct_change()  # pandas 内置方法
```

### 2.2 对数收益率 (Log Return)

**公式：**

$$
r_t = \ln(P_t / P_{t-1}) = \ln(P_t) - \ln(P_{t-1})
$$

**含义**：连续复利下的增长率。

**例子**：股票从 100 元涨到 105 元

$$
r = \ln(105/100) = \ln(1.05) \approx 0.04879 \approx 4.879
$$

**代码**：

```python
log_returns = np.log(df["Close"] / df["Close"].shift(1))
```

### 2.3 两者的区别与选择

| 特性               | 简单收益率                             | 对数收益率                          |
| ------------------ | -------------------------------------- | ----------------------------------- |
| 多期累计           | 连乘：$(1+R₁)(1+R₂)...(1+Rₙ) - 1$ | 直接相加：$r₁ + r₂ + ... + rₙ$ |
| 截面（多资产）加总 | ✅ 可以加权平均                        | ❌ 不能直接加                       |
| 数学性质           | 下界 -100%（不能亏超过本金）           | 可以是任意负数                      |
| 近似关系           | 收益率小时两者几乎相等                 | 收益率小时两者几乎相等              |

**经验法则**：

- **单资产时间序列分析** → 用对数收益率（方便累加、统计性质好）
- **多资产组合计算** → 用简单收益率（可以加权）
- **实际回测** → 用简单收益率（直接对应你的盈亏）

> 💡 **程序员类比**：简单收益率像 `for` 循环中累乘，对数收益率像把乘法变成加法（就像 `log` 把指数运算变成线性运算）。数学上更方便处理。

## 三、年化收益率 (Annualized Return)

日收益率很小（比如 0.05%），不直观。我们通常换算成年化收益率。

### A 股交易日

A 股一年大约 **242 个交易日**（美股约 252 个）。

### 公式

**从日简单收益率年化：**

$$
R_{annual} = (1 + R_{daily\_mean})^{242} - 1
$$

**从日对数收益率年化：**

$$
r_{annual} = r_{daily\_mean} \times 242
$$

> 💡 注意：不能简单地用 `日收益率 × 242`（算术平均会高估），应该用**几何平均**或等价的对数方法。

### 为什么用几何平均而不是算术平均？

假设连续两天的收益率是 +50% 和 -50%：

- 算术平均：(50% + (-50%)) / 2 = 0%（看起来没亏）
- 实际情况：100 → 150 → 75，实际亏了 25%！

几何平均才反映真实复利增长。

## 四、波动率 (Volatility)

### 4.1 定义

波动率 = 收益率的标准差，衡量价格的"颠簸程度"。

**日波动率：**

$$
\sigma_{daily} = \text{std}(r_1, r_2, \dots, r_n)
$$

**年化波动率：**

$$
\sigma_{annual} = \sigma_{daily} \times \sqrt{242}
$$

> 💡 为什么乘以 √242 而不是 242？这来自统计学中"独立随机变量方差可加性"：
> $Var(X₁+X₂+...+Xₙ) = n × Var(X)，所以 Std = √n × σ$。

### 4.2 波动率的含义

- 年化波动率 15% → 比较稳健的蓝筹股
- 年化波动率 30% → 成长股
- 年化波动率 50%+ → 高波动（如创业板、小盘股）

### 4.3 代码

```python
daily_vol = returns.std()
annual_vol = daily_vol * np.sqrt(242)
```

## 五、夏普比率 (Sharpe Ratio) ⭐

这是量化交易中**最重要的指标**，没有之一。

### 5.1 定义

$$
\text{Sharpe} = \frac{R_p - R_f}{\sigma_p}
$$

- `R_p`：策略的年化收益率
- `R_f`：无风险利率（通常取国债收益率，A 股常用 3% 即 0.03）
- `σ_p`：策略的年化波动率

### 5.2 含义

**每承担 1 单位风险（波动率），你能获得多少超额收益。**

| Sharpe    | 评价                   |
| --------- | ---------------------- |
| < 0       | 亏损                   |
| 0 ~ 0.5   | 差                     |
| 0.5 ~ 1.0 | 一般                   |
| 1.0 ~ 2.0 | 好                     |
| 2.0 ~ 3.0 | 非常好                 |
| > 3.0     | 可能有问题（过拟合？） |

> 💡 **关键直觉**：Sharpe = 1 意味着你赚的超额收益刚好等于你承受的波动。Sharpe = 2 意味着你每承受 1% 的波动就赚 2% 的超额。

### 5.3 实际计算注意

**在回测中，我们通常用日度 Sharpe 然后年化：**

$$
\text{Sharpe} = \frac{R_p - R_f}{\sigma_p}
$$

其中 `daily_excess_return = daily_return - rf_daily`，`rf_daily ≈ 0.03/242`。

## 六、最大回撤 (Maximum Drawdown) ⭐

### 6.1 定义

**最大回撤 = 从最高点到最低点的最大跌幅**

$$
\text{MaxDD} = \max_{t} \left[ \frac{P_t - \text{Peak}_t}{\text{Peak}_t} \right]
$$

直观地说：**你最多会亏多少？**

### 6.2 计算步骤

1. 算出累计收益曲线（净值曲线）
2. 在每个时间点，记录到目前为止的最高净值（running max）
3. 当前净值相对最高净值的跌幅 = drawdown
4. 取所有 drawdown 的最大值 = MaxDD

### 6.3 代码

```python
cumulative = (1 + returns).cumprod()
running_max = cumulative.cummax()
drawdown = (cumulative - running_max) / running_max
max_drawdown = drawdown.min()  # 注意是负数
```

### 6.4 为什么重要？

- Maximum Drawdown 衡量的是**最坏情况下的心理承受能力**
- 即使你的策略最终赚钱了，如果中间回撤了 50%，你可能早就止损了
- 很多机构投资者的回撤限制是 10%~20%

## 七、索提诺比率 (Sortino Ratio)

### 7.1 夏普比率的问题

Sharpe 用的是"总波动率"，但实际上**向上的波动不是风险，只有向下的波动才是风险**。

比如两个策略：

- 策略 A：收益率序列 +5%, +3%, +8%, +1%, +6% → 波动率不低，但全是赚钱
- 策略 B：收益率序列 +5%, -3%, +8%, -1%, +6% → 同样波动率，但有亏损

Sharpe 对他们的评价一样，但显然 A 更好。Sortino 就修正了这个问题。

### 7.2 定义

$$
\text{Sortino} = \frac{R_p - R_f}{\sigma_{\text{downside}}}
$$

其中 `σ_downside` 只用**负收益率**来计算标准差。

### 7.3 代码

```python
downside_returns = returns[returns < 0]
downside_std = downside_returns.std() * np.sqrt(242)
sortino = (annual_return - risk_free) / downside_std
```

## 八、卡尔玛比率 (Calmar Ratio)

### 8.1 定义

```
Calmar = R_annual / |MaxDD|
```

**年化收益 ÷ 最大回撤的绝对值**

### 8.2 含义

每承受 1% 的最大回撤，你能赚多少年化收益。

| Calmar | 评价               |
| ------ | ------------------ |
| < 1    | 差（赚的不够亏的） |
| 1 ~ 2  | 一般               |
| 2 ~ 3  | 好                 |
| > 3    | 很好               |

## 九、实践总结

### 指标一览表

| 指标       | 衡量什么         | 越大越好？           |
| ---------- | ---------------- | -------------------- |
| 年化收益率 | 赚了多少         | ✅                   |
| 年化波动率 | 颠簸程度         | ❌（越小越好）       |
| 最大回撤   | 最多亏多少       | ❌（绝对值越小越好） |
| 夏普比率   | 风险调整收益     | ✅                   |
| 索提诺比率 | 下行风险调整收益 | ✅                   |
| 卡尔玛比率 | 回撤调整收益     | ✅                   |
